# Combine for AI Extension

This is a simple utility extension for Visual Studio Code and Cursor to quickly combine the content of multiple files into the clipboard for use with AI prompts.

## Features

-   Select multiple files in the File Explorer or the Open Editors view.
-   Right-click and choose "Combine for AI".
-   The content of all selected files, each preceded by its relative path, is copied to your clipboard.

## Usage

1.  Select one or more files.
2.  Right-click on any of the selected files.
3.  Click "Combine for AI".
4.  Paste the result into your AI prompt.
